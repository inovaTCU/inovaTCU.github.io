inovaTCU v0.1

Controle mais efetivo, mais benefícios para o cidadão.

Inicialmente, vou apenas listar o conteúdo básico para a página de projetos e da galeria. O futuro site oficial terá como endereço: inovatcu.gov.br

Projetos

  • SecexPrevi: fiscalização contínua, automação de fluxo de trabalho de análise, dashboard de indicadores;
  • Auditoria Contínua: Rutgers (Miklos, ComprasNet);
  • Seinfra Ferrovia: geo referenciamento, traçado de ferrovias;

Galeria de Análises de Dados

  • LabContas: tutorial acesso, exemplos de análise
  • Mapas com dados em R
  • Material do curso de Mineração de Dados

Possíveis outras seções

  • referência a outros portais de inovação e govLabs: MaRS, MindLab.dk, SP inovaday, EtaLab, participa.br;

  • Parcerias: acordos de cooperação (acesso aos documentos)

In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
